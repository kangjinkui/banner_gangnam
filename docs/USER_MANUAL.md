# 현수막 관리 시스템 이용 매뉴얼

처음 사용하는 사용자가 정당 현수막, 공공 현수막, 집회시위 현수막을 조회하고 등록하고 관리할 수 있도록 정리한 안내서입니다.

## 1. 프로그램 개요

이 시스템은 강남구 현수막 정보를 지도와 목록으로 관리하는 웹 프로그램입니다.

- `정당 현수막`: 정당 소속 현수막을 관리합니다.
- `공공·집회시위`: 공공 현수막과 집회시위 현수막을 관리합니다.
- `지도`: 현수막 위치를 지도에서 확인합니다.
- `목록`: 현수막을 표 또는 카드 형태로 확인하고 검색합니다.
- `통계`: 현황을 요약해서 확인합니다.
- `만료`: 종료일이 지난 현수막을 확인합니다.

## 2. 처음 접속했을 때

메인 화면 상단에는 다음 메뉴가 있습니다.

- `정당 현수막`
- `공공·집회시위`
- `로그인` 또는 사용자 정보
- `등록` 버튼

모바일에서는 상단 헤더가 2줄로 보일 수 있으며, `정당 현수막`과 `공공·집회시위` 버튼은 가로 스크롤로도 확인할 수 있습니다.

## 3. 로그인 방법

1. 화면 오른쪽 상단의 `로그인` 버튼을 누릅니다.
2. 이메일과 비밀번호를 입력합니다.
3. 로그인에 성공하면 사용자 이메일 또는 관리 메뉴가 표시됩니다.

권한에 따라 가능한 기능이 다릅니다.

- `일반 사용자`: 데이터 조회, 본인 계정 관리
- `관리자`: 현수막 등록, 수정, 삭제, 정당 관리, 사용자 관리

## 4. 화면 구성 이해하기

메인 화면은 보통 아래 순서로 구성됩니다.

1. 상단 헤더
2. 현황 카드
3. 탭 메뉴 (`지도`, `목록`, `통계`, `만료`)
4. 실제 내용 영역

현황 카드에서는 전체 건수, 활성 건수, 만료 건수 등을 한눈에 볼 수 있습니다.

## 5. 정당 현수막 조회하기

1. 상단의 `정당 현수막` 버튼을 누릅니다.
2. 아래 탭 중 원하는 화면을 선택합니다.

- `지도`: 위치 중심으로 확인
- `목록`: 검색과 필터 중심으로 확인
- `통계`: 수량과 분포 확인
- `만료`: 종료된 현수막 확인

### 목록에서 할 수 있는 일

- 주소, 문구 등으로 검색
- 정당별 필터
- 상태별 필터
- `엑셀 내보내기`로 파일 다운로드
- 항목을 눌러 상세 정보 확인

## 6. 공공·집회시위 현수막 조회하기

1. 상단의 `공공·집회시위` 버튼을 누릅니다.
2. 필요하면 상단 필터에서 아래 중 하나를 선택합니다.

- `전체`
- `공공`
- `집회시위`

3. 아래 탭에서 `지도`, `목록`, `통계`, `만료` 중 원하는 화면을 선택합니다.

### 목록에서 할 수 있는 일

- 검색어 입력
- 상태 필터
- 유형별 확인
- `엑셀 내보내기`
- 현수막 상세 확인

## 7. 정당 현수막 등록하기

관리자 권한이 있는 경우 상단에 `등록` 버튼이 보입니다.

1. `정당 현수막` 화면에서 `등록` 버튼을 누릅니다.
2. 아래 항목을 입력합니다.

- 정당 선택
- 현수막 문구
- 설치 주소
- 시작일
- 종료일
- 메모
- 사진 업로드

3. 주소를 입력하면 시스템이 위치와 행정동을 확인합니다.
4. 필요하면 현재 위치 기능을 사용해 주소를 가져올 수 있습니다.
5. 내용을 확인한 뒤 등록을 완료합니다.

### 사진 업로드 주의사항

- 이미지 파일만 업로드할 수 있습니다.
- 큰 파일은 업로드가 실패할 수 있으므로 촬영 후 적절한 크기로 사용하는 것이 좋습니다.

## 8. 공공·집회시위 현수막 등록하기

1. `공공·집회시위` 화면에서 `등록` 버튼을 누릅니다.
2. 먼저 현수막 유형을 선택합니다.

- `공공 현수막`
- `집회시위 현수막`

3. 공통 항목을 입력합니다.

- 설치 주소
- 현수막 문구
- 시작일
- 종료일
- 사진
- 메모

4. 유형에 따라 추가 항목을 입력합니다.

- 공공 현수막: `부서명`
- 집회시위 현수막: `게시자명`

5. `등록` 버튼을 눌러 저장합니다.

## 9. 현수막 상세 보기, 수정, 삭제

목록이나 지도에서 현수막을 누르면 `현수막 상세` 창이 열립니다.

상세 화면에서 확인할 수 있는 정보:

- 현수막 이미지
- 정당 또는 유형
- 현수막 문구
- 주소와 행정동
- 시작일, 종료일
- 메모
- 등록일, 수정일

관리자 권한이 있으면 다음 버튼이 보일 수 있습니다.

- `수정`
- `삭제`

### 수정 방법

1. `수정` 버튼을 누릅니다.
2. 내용을 변경합니다.
3. `저장` 또는 완료 버튼을 눌러 반영합니다.

### 삭제 방법

1. `삭제` 버튼을 누릅니다.
2. 확인 메시지를 읽고 승인합니다.
3. 삭제 후 목록에서 사라집니다.

주의: 삭제는 되돌릴 수 없을 수 있으므로 신중히 진행하세요.

## 10. 내 정보 관리

로그인 후 사용자 아이콘 또는 이메일을 누르면 `내 정보` 화면으로 이동할 수 있습니다.

이 화면에서 할 수 있는 일:

- 로그인한 이메일 확인
- 사용자 권한 확인
- 가입일 확인
- 비밀번호 변경

### 비밀번호 변경 방법

1. 현재 비밀번호 입력
2. 새 비밀번호 입력
3. 새 비밀번호 확인 입력
4. 저장

비밀번호는 최소 6자 이상이어야 합니다.

## 11. 관리자 기능

관리자에게는 추가 메뉴가 보일 수 있습니다.

### 11.1 정당 관리

`정당 관리` 메뉴에서 아래 작업이 가능합니다.

- 새 정당 추가
- 정당명 수정
- 정당 색상 지정
- 활성화/비활성화
- 정당 삭제

정당 삭제 시 연관 데이터에 영향이 있을 수 있으므로 주의하세요.

### 11.2 사용자 관리

`사용자 관리` 메뉴에서 아래 작업이 가능합니다.

- 전체 사용자 조회
- 이메일 검색
- 관리자 수 / 일반 사용자 수 확인
- 사용자 권한 변경

권한 변경은 즉시 서비스 이용 범위에 영향을 줍니다.

## 12. 자주 사용하는 업무 예시

### 예시 1. 새 정당 현수막 등록

1. 로그인
2. `정당 현수막` 선택
3. `등록` 클릭
4. 정당, 주소, 문구, 기간, 사진 입력
5. 저장
6. 목록 또는 지도에서 등록 결과 확인

### 예시 2. 공공 현수막만 확인

1. `공공·집회시위` 선택
2. 상단 유형 필터에서 `공공` 선택
3. `목록` 또는 `지도` 탭 선택

### 예시 3. 만료 현수막 확인

1. 원하는 화면으로 이동
2. `만료` 탭 선택
3. 종료일이 지난 현수막을 확인

## 13. 문제 해결

### 데이터가 보이지 않을 때

- 먼저 화면을 새로고침합니다.
- 로그인 상태가 유지되는지 확인합니다.
- 인터넷 연결 상태를 확인합니다.
- 계속 보이지 않으면 관리자에게 문의합니다.

### 로그인되지 않을 때

- 이메일과 비밀번호를 다시 확인합니다.
- 비밀번호를 잊은 경우 관리자에게 계정 초기화를 요청합니다.

### 주소가 인식되지 않을 때

- 도로명 주소 또는 보다 자세한 주소로 다시 입력합니다.
- 건물명 대신 실제 주소를 입력해봅니다.

### 사진 업로드가 실패할 때

- 이미지 형식인지 확인합니다.
- 파일 용량을 줄인 뒤 다시 시도합니다.
- 모바일에서는 촬영 후 원본이 너무 크면 업로드가 느릴 수 있습니다.

## 14. 사용 시 주의사항

- 현수막 정보는 실제 설치 위치 기준으로 정확히 입력하세요.
- 시작일과 종료일을 가능한 한 정확히 입력하세요.
- 삭제 전에는 반드시 대상이 맞는지 확인하세요.
- 관리자 권한 변경은 신중하게 진행하세요.

## 15. 문의 전 확인하면 좋은 정보

문제가 발생했을 때 아래 내용을 함께 전달하면 빠르게 확인할 수 있습니다.

- 어느 메뉴에서 문제가 발생했는지
- 어떤 버튼을 눌렀는지
- 입력한 주소 또는 검색어
- 오류 문구가 있었는지
- PC인지 모바일인지
- 문제가 발생한 시간
